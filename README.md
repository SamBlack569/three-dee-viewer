# Three Dee Viewer (Learning Project)

This repository is a personal learning project focused on **Three.js** and viewer development.

The main purpose is to learn and deepen my understanding of web viewers, then apply that learning directly in my current job.

## Current Scope

- Build and understand a basic Three.js scene
- Experiment with camera, lighting, materials, and rendering setup
- Create a solid foundation for a production-oriented viewer

## Next Goals

- Add support for **IFC** files
- Add support for **Frags** (That Open Fragment)
- Explore workflows for BIM visualization in the browser

## Tech Stack

- [Three.js](https://threejs.org/)
- [Vite](https://vitejs.dev/)
- TypeScript

## Run Locally

```bash
npm install
npm run dev
```

Then open the local URL shown in the terminal (usually `http://localhost:5173`).
